# mbed-gy88
This is my second attempt at getting the GY-88 sensor chip running on the LPC1768.

## Dev Environment
- Mac OS X 10.10
- gcc version 4.9.3 20141119 (release) [ARM/embedded-4_9-branch revision 218278] (GNU Tools for ARM Embedded Processors)

## Build

```
make
```

## Deploy

```
make deploy
```
